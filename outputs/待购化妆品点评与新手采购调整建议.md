# 待购化妆品点评与新手采购调整建议

> 适用对象：化妆新手，目标是先完成干净、自然、日常可出门的基础妆，而不是一步到位做复杂修容和完整眼妆。

## 1. 总结论

你目前准备买的四件是：

1. **M.A.C 小金盒高光修容双色盘**
2. **3CE 九色眼影 #橘棕盘**
3. **3CE 单色腮红 #裸粉色**
4. **YSL 女神粉底液 B10**

我的判断：

| 商品 | 建议 | 原因 |
|---|---|---|
| M.A.C 小金盒高光修容双色盘 | **暂缓** | 修容和高光是进阶项，新手容易显脏、显毛孔、鼻影不自然 |
| 3CE 九色眼影 #橘棕盘 | **可买但不急** | 颜色经典，但眼影不是新手第一优先级，橘棕对肿眼泡可能显肿 |
| 3CE 单色腮红 #裸粉色 | **可以保留** | 腮红对新手友好，能快速提升气色，单色比多色盘更简单 |

> **保留腮红；粉底先试色；眼影可以后置；M.A.C 小金盒先别急。**

## 2. 单品点评

## 2.1 M.A.C 小金盒高光修容双色盘

### 放在哪个化妆步骤？

- 对应步骤：**修容 / 高光**
- 使用位置：鼻影、侧脸、颧骨、鼻梁、眉骨等
- 新手优先级：**低**

### 优点

- 包含高光和修容，一盘两个功能；
- Omega 灰棕修容比很多偏红偏橘的修容更自然；
- 生姜高光对黄皮相对友好；
- 颜值高，适合后续进阶。

### 风险

- 新手修容容易画重，导致脸脏；
- 鼻影容易变成两条灰线；
- 高光如果扫到毛孔、纹理、闭口明显的位置，会放大瑕疵；
- 日常淡妆里，修容和高光不是必须步骤。

### 建议

**暂缓购买。**

等你能稳定完成以下步骤之后再买更合适：

1. 底妆不卡粉、不假面；
2. 眉毛自然；
3. 腮红不会下手太重；
4. 眼妆不显脏。

---

## 2.2 3CE 九色眼影 #橘棕盘

### 放在哪个化妆步骤？

- 对应步骤：**眼妆**
- 使用位置：眼皮打底、眼尾加深、卧蚕/眼中提亮
- 新手优先级：**中**

### 优点

- 橘棕、大地色系日常，适合韩系/暖调妆容；
- 一盘里通常有打底色、过渡色、加深色和亮片色；
- 可以练习基础眼影层次；
- 颜色风格比较适合亚洲妆容。

### 风险

- 如果你是肿眼泡，橘色、红棕色、珠光色可能显肿；
- 九色盘对新手有选择压力；
- 眼影晕染需要练习，刚开始容易显脏；
- 新手早期其实可以先用睫毛夹 + 睫毛膏提升眼神，不必急着画完整眼影。

### 建议

**可以留，但不是第一优先级。**

如果你喜欢暖橘、南瓜、韩系日常妆，可以买；但建议先从最简单的三色法开始：

1. 浅色大面积打底；
2. 中间色加深眼尾和双眼皮褶皱；
3. 少量亮片点在眼中或卧蚕。

## 2.3 3CE 单色腮红 #裸粉色

### 放在哪个化妆步骤？

- 对应步骤：**腮红**
- 使用位置：苹果肌、颧骨外侧、面中等
- 新手优先级：**高**

### 优点

- 单色腮红比多色盘更简单；
- 裸粉色日常，不容易太夸张；
- 能快速提升气色，避免底妆后脸色发白；
- 比修容、高光更容易上手。

### 风险

- 如果肤色偏黄偏暗，太浅的裸粉可能不显色或显灰；
- 如果脸颊本身泛红明显，粉色腮红可能叠加红感；
- 浅粉色有一定膨胀感，下手太重可能显脸肿。

### 建议

**可以保留。**

下单前建议确认具体色号：

- **MONO PINK**：偏温柔裸粉，更适合白皮/浅肤色；
- **NUDE PEACH**：裸米桃，更适合黄皮；
- **ROSE BEIGE**：玫瑰米色，日常更稳。

如果你的肤色不是很白，优先考虑裸桃、米杏、玫瑰豆沙类颜色。

## 2.4 YSL 女神粉底液 B10

### 放在哪个化妆步骤？

- 对应步骤：**底妆**
- 使用位置：全脸均匀肤色
- 新手优先级：**高，但必须试色**

### 优点

- 粉底是新手底妆核心品类；
- 色号合适时，对整体妆容提升很明显；
- 比眼影、修容更能决定“妆干不干净”。

### 最大风险

**B10 可能偏白。**

粉底不是越白越好。新手最容易犯的错误就是买太白，结果出现：

- 脸和脖子色差明显；
- 妆感假面；
- 暗沉后发灰；
- 越看越像戴面具。

### 试色方法

不要只涂手背。正确试色位置是：

1. 涂在**下颌线到脖子的交界处**；
2. 看它和脖子是否自然融合；
3. 等 10-30 分钟，看是否氧化变暗；
4. 在自然光下看，不要只看商场灯光；
5. 如果脸亮一截、脖子暗一截，就不要买这个色号。

### 色号判断

| 肤色情况 | B10 是否合适 |
|---|---|
| 冷白皮 / 黄一白 | 可以考虑 |
| 自然肤色 / 黄二白 | 可能偏白，建议试 B20 或自然色号 |
| 偏黄暗沉 | 不建议为了提亮硬买太白色号 |

### 建议

**粉底可以买，但 B10 不要盲买正装。先试色，最好要小样或去柜台试。**

## 3. 你这套清单缺少的基础品类

你现在准备买的是：粉底、腮红、眼影、修容高光。

但对新手来说，真正第一批更应该补齐这些：

| 品类 | 优先级 | 为什么需要 |
|---|---:|---|
| 卸妆产品 | 高 | 买了粉底、眼影、腮红后必须卸干净 |
| 眉笔或眉粉 | 高 | 眉毛对精神感影响很大，比眼影更基础 |
| 睫毛夹 | 高 | 夹翘睫毛后眼睛更有神，性价比很高 |
| 睫毛膏 | 高 | 比复杂眼影更适合新手早期提升眼妆 |
| 散粉或粉饼 | 高 | 粉底后需要定妆，尤其是 T 区、鼻翼、眼下 |
| 美妆蛋 / 粉扑 | 高 | 没有工具，粉底很难拍匀 |
| 唇膏 / 唇釉 | 中高 | 提气色，和腮红一起决定整体气色 |
| 遮瑕 | 中 | 有明显黑眼圈、痘印、鼻翼泛红时再买 |
| 眼线 | 中低 | 容易翻车，可等睫毛和眼影练熟后再买 |
| 修容 / 高光 | 低 | 进阶步骤，先不急 |

---

## 4. 推荐调整后的第一批采购顺序

### 第一优先级：先把日常妆跑通

1. 粉底 / 气垫 / 肤色润色产品：**YSL B10 试色后再定**；
2. 腮红：**3CE 裸粉色可以保留**；
3. 眉笔或眉粉；
4. 睫毛夹；
5. 睫毛膏；
6. 散粉或粉饼；
7. 卸妆产品；
8. 美妆蛋或粉扑；
9. 日常唇膏/唇釉。

### 第二优先级：有余力再练

1. **3CE 九色眼影 #橘棕盘**；
2. 遮瑕；
3. 眼线笔。

### 第三优先级：进阶后再买

1. **M.A.C 小金盒高光修容双色盘**；
2. 高光刷、修容刷；
3. 多色修容/复杂眼影盘。

## 5. 最终建议

如果只针对你这四件，我的购买排序是：

1. **YSL 女神粉底液 B10：重要，但必须先试色**
2. **3CE 单色腮红 #裸粉色：可以买**
3. **3CE 九色眼影 #橘棕盘：可以，但不急**
4. **M.A.C 小金盒高光修容双色盘：先别买**

最稳的新手路线是先做好：

> 底妆干净 → 眉毛自然 → 睫毛有神 → 腮红提气色 → 唇色协调。

等这几个步骤稳定之后，再进入眼影、眼线、修容、高光。
